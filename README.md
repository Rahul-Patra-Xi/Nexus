## AI assistant (edge function)

The `ai-assistant` Supabase Edge Function calls an OpenAI-compatible Chat Completions API. Set these secrets for deployed functions (for example `supabase secrets set`):

- `OPENAI_API_KEY` — required
- `OPENAI_BASE_URL` — optional, default `https://api.openai.com/v1`
- `OPENAI_MODEL` — optional, default `gpt-4o-mini`

Local development: configure the same variables in your Supabase CLI environment when serving functions.
